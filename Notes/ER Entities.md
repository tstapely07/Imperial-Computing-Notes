> [!Abstract] Definition
> An [[ER Modelling|ER]] **entity** $E$ represents a set of objects which are conceptually the same type of thing.

These are usually **nouns**, such as `person` or `department`.
**Entities** are represented by **rectangles** in **ER diagrams**.

When mapping [[ER Modelling|ER]] to a [[relational schema]]:
* Each **entity** $E$ maps to a **table** $R_E$.
