> [!Abstract] Definition
> An **adjacency list** is an **array** of **linked lists**.
> For each [[Nodes|node]], we store a list of **adjacent nodes.**

**Multiple** [[Arcs|arcs]] result in **multiple entries**.
Each [[Arcs|arc]] gets entered twice, with the exception of **loops**, which are only entered **once**.
