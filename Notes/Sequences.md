> [!Abstract] Definition
> A **sequence** $(a_n)_{n\ge 1}$ of real numbers is a **function** that maps the positive integers to the real numbers ($f:\mathbb{N}^+\to \mathbb{R}$).
> It produces an ordered list of terms: $a_1,a_2,a_3,\dots$

> [!Example] Examples
> * $a_n=\frac{1}{n}\Rightarrow 1, \frac{1}{2}, \frac{1}{3},\dots$ (**converges** to $0$)
> * $a_n=n\Rightarrow 1,2,3,\dots$ (**converges** to $\infty$) (note some material may say **diverges** to $\infty$)
> * $a_n=1+\frac{(-1)^n}{n}\Rightarrow 0, \frac{3}{2}, \frac{2}{3}, \frac{5}{4},\dots$ (**oscillates** about $1$, but **converges** to $1$)

> [!Abstract] Definitions
> Let $(a_n)_{n\ge1}$ be a **sequence** of real numbers.
> * The sequence is **increasing** if for all $n\ge 1$, $a_n\le a_{n+1}$.
> * The sequence is **strictly increasing** if for all $n\ge 1$, $a_n\lt a_{n+1}$.
> * The sequence is **decreasing** if for all $n\ge 1$, $a_n\ge a_{n+1}$.
> * The sequence is **strictly decreasing** if for all $n\ge 1$, $a_n\gt a_{n+1}$.
> 

> [!Abstract] Definitions
> A sequence is **monotonic** if it is either **increasing** or **decreasing**.
> A sequence is **strictly monotonic** if it is either **strictly increasing** or **strictly decreasing**.
