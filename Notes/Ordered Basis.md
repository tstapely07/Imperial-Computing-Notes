An **ordered basis** is a [[basis of a subspace]] where the specific sequence of the [[vectors]] is fixed.

This **ordered basis** allows us to define **coordinates**.

This fixed order is required to build:
* Coordinate vectors - see [[basis change]].
* Matrix representations - see [[matrix representation of linear mappings]].

The most common example is the [[standard ordered basis]].
