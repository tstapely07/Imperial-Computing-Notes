> [!Abstract] Definition
> A **register** is a bank of $n$ [[edge-triggered D-type flip-flop|edge-triggered D-type flip-flops]] that share a common [[Clocks|clock]]. 
> * This means that all **bits** are set at the exact same time, allowing a **register** to store an $n$-bit [[binary]] number.

Common uses of registers include:
* The **address register**, **data register**, **program counter**, and other **registers** within the **CPU**.
* The **state register** used within a [[Synchronous Circuits|synchronous circuit]] to store the state of a [[Finite State Machines|finite state machine]].
