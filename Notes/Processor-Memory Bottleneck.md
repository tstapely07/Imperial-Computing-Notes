Modern **CPUs** process data incredibly fast, but [[x86 Memory Layout|main memory]] is far away from the **CPU** and so is slow to access.
* ![[Pasted image 20260420165626.png|312]]
To solve this, there are small, fast **memory chips**, known as [[x86 Cache|cache]] built directly into the **CPU**.
