A **traversal algorithm** starts at some [[Nodes|node]].
It visits **every node**, using some [[Paths|path]].
It is allowed to **backtrack**.
It will reach **every node**, if the **graph** is [[Connectedness|connected]].

Two **traversal algorithms** are:
* [[Depth-First Search (DFS)]]
* [[Breadth-First Search (BFS)]]
Both **algorithms** will traverse all the **nodes** in a **connected graph**, but the order of visiting is different.
* Sometimes, either procedure will do, but other times one procedure is to be preffered.
