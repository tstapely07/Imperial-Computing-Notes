A **helper function** is a function that performs a **specific sub-task** to support the **primary function**.
If a helper function is only useful to a **single function**, it should be defined within a [[Where Clauses|where clause]].

>[!Tip]
>**Helper functions** can hide an [[Accumulating Parameters|accumulating parameter]], which is used in making a function [[Tail Recursion|tail recursive]].
